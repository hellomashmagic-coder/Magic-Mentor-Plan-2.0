## Getting Started

### Local Development
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

### Features
- **Access Approval System**: Integrated request/approval workflow for user access.
- **Security Deterrents**: Auto-blur on focus loss, right-click protection, and confidential watermarks.
- **Dynamic Curriculum**: 30-day mentorship plans for Level 1, 2, and 3 students.
